# v2.2.0 Visualization & Tabbed Interface Design

**Date:** 2025-02-13
**Version:** 2.2.0
**Status:** Design Complete, Ready for Implementation

## Executive Summary

v2.2.0 transforms the results interface from a long scrolling page into an organized tabbed workspace with rich Recharts visualizations. The goal is to serve two user personas:

1. **Primary user (business owner)**: "Just tell me what to do" â†’ Tab 1 provides the answer in 10 seconds
2. **Secondary user (advisor/accountant)**: "Show me the proof" â†’ Tabs 2-3 provide full comparison and detailed breakdowns

### Key Value Proposition

The calculator answers: **"Should I pay myself salary or dividends, and if so how much?"**

- **Tab 1 (Recommended)**: Immediate answer with actionable year-by-year plan
- **Tab 2 (Compare All)**: Validate the recommendation by seeing all 3 strategies side-by-side
- **Tab 3 (Details)**: Full tax breakdowns and granular charts for accountant scrutiny
- **Tab 4 (Export)**: PDF, email, and share link consolidated in one place

---

## Problem Being Solved

### Current Issues
1. **Too much scrolling**: Users must scroll up/down to compare different sections
2. **Information overload**: All data shown at once, overwhelming casual users
3. **Hidden value**: Strategy comparison charts don't exist yet, making it hard to visualize trade-offs
4. **Unclear action**: No clear "here's what to do next" guidance

### Solution
- **Tabbed interface**: Organize content by user intent (recommendation â†’ comparison â†’ details â†’ export)
- **Progressive disclosure**: Show simple answer first, details on-demand
- **Rich visualizations**: Multi-strategy charts showing trajectories, breakdowns, and comparisons
- **After-tax wealth transparency**: Show both PWL-optimal scoring AND practical after-tax reality

## Data Structure Updates

### 1. After-Tax Wealth Calculation

Add to `strategyComparison.ts`:

```typescript
interface AfterTaxWealthScenarios {
  atCurrentRate: number;      // Using average marginal rate
  atLowerRate: number;        // marginal - 10%, min 20%
  atTopRate: number;          // Top provincial rate
  assumptions: {
    currentRRSPWithdrawalRate: number;
    lowerRRSPWithdrawalRate: number;
    topRRSPWithdrawalRate: number;
    corpLiquidationRate: number;  // Fixed at 40%
  };
}

interface StrategyResult {
  // ... existing fields
  trueAfterTaxWealth: AfterTaxWealthScenarios;
}
```

**Calculation:**
```typescript
function calculateAfterTaxWealth(
  summary: ProjectionSummary,
  province: ProvinceCode,
  averageMarginalRate: number
): AfterTaxWealthScenarios {

  const topRate = getTopProvincialRate(province);
  const lowerRate = Math.max(averageMarginalRate - 0.10, 0.20);
  const corpLiquidationRate = 0.40; // Conservative (non-eligible dividends)

  const baseWealth =
    summary.totalAfterTaxIncome +  // Already taxed
    summary.tfsaContributions;     // Tax-free forever

  return {
    atCurrentRate: baseWealth +
      (summary.rrspBalance * (1 - averageMarginalRate)) +
      (summary.finalCorporateBalance * (1 - corpLiquidationRate)),

    atLowerRate: baseWealth +
      (summary.rrspBalance * (1 - lowerRate)) +
      (summary.finalCorporateBalance * (1 - corpLiquidationRate)),

    atTopRate: baseWealth +
      (summary.rrspBalance * (1 - topRate)) +
      (summary.finalCorporateBalance * (1 - corpLiquidationRate)),

    assumptions: { currentRRSPWithdrawalRate, lowerRRSPWithdrawalRate, topRRSPWithdrawalRate, corpLiquidationRate }
  };
}
```

## Responsive Design

### Tab Navigation

**Desktop (>768px):**
- Horizontal tabs with underline indicator
- All tabs visible at once

**Mobile (<768px):**
- Dropdown/select menu
- Shows current tab name + chevron
- Saves vertical space

### Charts

**Desktop:**
- 2-column grid where appropriate (e.g., Tax Breakdown + Comp Mix side-by-side)
- Full-width for time-series charts

**Tablet/Mobile:**
- Single column, full-width
- Horizontal scroll for wide tables
- Consider "tap to expand" for full-screen chart view
